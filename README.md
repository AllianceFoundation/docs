# docs
Официальная документация Alliance Foundation  

Этот репозиторий содержит официальную документацию Некоммерческого фонда Alliance Foundation.  
Фонд выполняет функции стандартизации, контроля, аккредитации и надзора в рамках экосистемы стейблкоина RUBA.  
Alliance Foundation не является операционным центром по выпуску виртуальных активов, банком или иным финансовым институтом, а также не осуществляет эмиссию токенов.

Фонд обеспечивает методологическую, организационную и техническую поддержку экосистемы, отвечает за развитие архитектуры, прозрачность процессов и соблюдение принципов безопасного обращения токена.

В репозитории будет представлен публичный whitepaper стейблкоина RUBA, предназначенный для ознакомления всех заинтересованных сторон.  
Публичный whitepaper включает следующие разделы:
• краткое резюме о стейблкоине RUBA  
• процессы эмиссии и сжигания  
• модель обеспечения  
• техническая инфраструктура экосистемы RUBA, включая платформы и смарт-контракты  
• токен управления rDAO для реализации мультиэмитентной модели стейблкоина RUBA

Документы в этом репозитории формируют нормативно-методическую базу проекта и обеспечивают прозрачность для регуляторов, партнёров, аудиторов и участников DAO.

Документация будет обновляться и дополняться по мере развития проекта и появления новых требований.
